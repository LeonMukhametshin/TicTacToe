# TicTacToe (Unity + LeoECS)

> Крестики‑нолики» в формате 6×6 с условием победы 3 в ряд. Проект показывает работу с игровыми состояниями, ECS и UI в Unity.

![Cover](Docs/cover.png)

## Технологии
- **Unity 6000.0.32f1**
- **C#**
- **LeoECS**
- **Unity UI**

## Архитектура
- **ECS**: компоненты состояния, события и системы обработки.
- **Разделение по сценам**: Main Menu и Gameplay имеют независимые точки старта.
- **UI как представление**: визуальные компоненты синхронизируются с ECS-состоянием.
- **Конфигурация через данные**: параметры игры задаются через настройки и передаются между сценами.

## Что реализовано
- Игровое поле 6х6 и проверка победы (3 символа в ряд).
- Поддержка нескольких игроков.
- Настройка параметров матча из Main Menu:
  - количество игроков;
  - время на ход.
- Таймер хода в игровом HUD и обработка завершения времени.
- Экраны результата (победа и ничья).
- Обновлённый UI и набор ассетов.
